Create Two Posts

A post in here4.me is a chunk of data associated with a given context. The type and format of the data are entirely up to the content producer. Possible post data formats could be the following:

  • Text
  • Binary data
  • URL
  • JSON object
  • XML document
  • HTML document

The only limitation of post data is it may not exceed 5K.

The content producer is responsible for developing the user interface for posting data. The user interface should include inputs and controls for post data and additional contextual information such as when the post is available or who can view it. The here4.me platform provides the user interface for selecting a location context.

For this demo, we are going to create very simple posts. The posts will have two pieces of data:

  • Text message for the post
  • Text message for the QR code

The user may optionally specify a time when the post is available. To get started, click the Enable Post Button and tap the Post icon in the navigation bar below.
