First, let's give our post a title. The title is the label that will show up under the post icon when the post is available.

Title

Now, let's define what content we want this post to have. This is a simple demo, so we will add a short text message. As stated before, the content can be as simple or complex as the content producer desires.

Post Message

Finally, we will define what content we want to show when the demo QR code is scanned. Keep in mind that not every site will require QR codes. QR codes have been added to this demo to show the QR code feature.

QR Code Message

Now we need to select a location where the post will be available. By default here4.me provides the functionality to search for and select locations by address. Please select an address different from your current one but also one you can quickly walk to. You can also search for an address or place by using the search box. For example, if you type Safeway and tap the search button, all the Safeway's near you will show up.

We now need to calculate the bounding box around the address provided. At this time, here4.me is not using a premium geocoding plan. Unfortunately, this means we are currently throttling down geocoding API calls. So this may take a minute. Please wait till 100% complete.
