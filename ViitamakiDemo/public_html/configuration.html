Set Personal QR Code

We will use this demo's configuration view to set the user's site QR code found on the QR Code view .

Users's Site QR Code Content

Once set another user may scan the QR Code from your mobile device to receive the content. You may also tap the QR Code to copy a URL you can send via text, email, or social media platforms. This is another way users can pass information from one user to another using here4.me.

To continue, tap the QR Code icon in the navigation bar below. After reviewing your site QR Code, tap the Viitamaki Demo icon in the navigation bar below to complete the demo.
